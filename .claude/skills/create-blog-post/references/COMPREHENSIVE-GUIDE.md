# Comprehensive Guide / Pillar Content Template (2026 Standard)

Use this template for definitive guides, pillar pages, and comprehensive resources (2,500+ words). These pages serve as the hub for topic clusters.

## Output Requirement

All content must be written in valid Markdown and be publish-ready.

## Frontmatter Template

```yaml
---
title: "Complete Guide to [Topic]: Everything You Need to Know in [Year]"
slug: ["the same as filename"]
description: "The definitive guide to [topic]. Learn from fundamentals to advanced techniques with practical examples and downloadable resources."
seoDescription: "The definitive guide to [topic]. Learn from fundamentals to advanced techniques with practical examples and downloadable resources."
faqs:
  - question: "What is [topic]?"
    answer: "Concise definition in 50-100 words..."
  - question: "Why is [topic] important in [year]?"
    answer: "Explanation of relevance..."
  - question: "How do I get started with [topic]?"
    answer: "First steps explanation..."
  - question: "What are the best tools for [topic]?"
    answer: "Tool recommendations..."
  - question: "How long does it take to learn [topic]?"
    answer: "Timeline explanation..."
draft: false
---
```

## Checklist Before Publishing

### SEO
- [ ] Word count is 2,500+ words
- [ ] Title is under 60 characters with primary keyword
- [ ] Meta description (seoDescription) is 150-160 characters
- [ ] URL slug is descriptive and short

### AEO/GEO
- [ ] First paragraph directly answers "what is [topic]?" (50-100 words)
- [ ] All H2s formulated as questions where possible
- [ ] Minimum 5 FAQs in frontmatter (pillar pages need more)
- [ ] Paragraphs are ~160 characters max

### E-E-A-T
- [ ] Statistics have sources cited
- [ ] Author with credentials assigned
- [ ] `lastUpdated` in frontmatter
- [ ] Original insights and firsthand experience included

### Topic Cluster
- [ ] This is clearly a pillar page (comprehensive topic)
